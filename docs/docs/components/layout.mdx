# Layout 布局组件

一个灵活的布局组件，支持多种布局模式，包括头部、侧边栏、主内容和底部区域的组合。

## 功能特性

- ✅ 支持 6 种布局模式（B、HB、HAB、HBF、AB、BF）
- ✅ 侧边栏可折叠功能
- ✅ 响应式设计
- ✅ 支持全高度布局
- ✅ 完整的 TypeScript 类型支持

## 布局模式说明

| 模式 | 说明 | 包含区域 |
|------|------|----------|
| B | Body | 主内容 |
| HB | Header + Body | 头部 + 主内容 |
| HAB | Header + Aside + Body | 头部 + 侧边栏 + 主内容 |
| HBF | Header + Body + Footer | 头部 + 主内容 + 底部 |
| AB | Aside + Body | 侧边栏 + 主内容 |
| BF | Body + Footer | 主内容 + 底部 |

## 基础用法

### 基础布局（HB 模式）

```tsx
import { Layout } from '@shrimp-design-react/components'

<Layout 
  mode="HB"
  header={
    <div style={{ padding: '16px', background: '#f0f0f0' }}>
      头部内容
    </div>
  }
>
  <div style={{ padding: '16px' }}>
    主内容区域
  </div>
</Layout>
```

**实际效果：**

<div style={{ border: '1px solid #e5e5e5', borderRadius: '4px', overflow: 'hidden', margin: '16px 0' }}>
  <Layout 
    mode="HB"
    fullHeight={false}
    header={
      <div style={{ padding: '16px', background: '#f0f0f0', textAlign: 'center' }}>
        <strong>头部内容</strong>
      </div>
    }
  >
    <div style={{ padding: '16px', minHeight: '200px' }}>
      主内容区域
    </div>
  </Layout>
</div>

### 受控的折叠状态

```tsx
import { Layout } from '@shrimp-design-react/components'
import { useState } from 'react'

function App() {
  const [collapsed, setCollapsed] = useState(false)

  return (
    <Layout 
      mode="HAB"
      isCollapse={collapsed}
      onCollapseChange={setCollapsed}
      header={<div>头部</div>}
      aside={<div>侧边栏</div>}
    >
      <div>主内容</div>
    </Layout>
  )
}
```

**实际效果：**

<div style={{ border: '1px solid #e5e5e5', borderRadius: '4px', overflow: 'hidden', margin: '16px 0', height: '300px' }}>
  <ControlledLayoutExample />
</div>

## API

### LayoutProps

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| mode | 布局类型 | `'B' \| 'HB' \| 'HAB' \| 'HBF' \| 'AB' \| 'BF'` | `'HB'` |
| isCollapse | 是否折叠（受控） | `boolean` | - |
| isCanCollapse | 是否可以折叠 | `boolean` | `true` |
| fullHeight | 是否全高 | `boolean` | `true` |
| asideBorder | 是否显示边框 | `boolean` | `true` |
| asideWidth | 侧边栏宽度 | `string \| number` | `300` |
| header | 头部内容 | `ReactNode` | - |
| aside | 侧边栏内容 | `ReactNode` | - |
| footer | 底部内容 | `ReactNode` | - |
| children | 主内容 | `ReactNode` | - |
| onCollapseChange | 折叠状态改变回调 | `(collapsed: boolean) => void` | - |

## 使用示例

### 禁用折叠功能

```tsx
import { Layout } from '@shrimp-design-react/components'

<Layout 
  mode="AB"
  isCanCollapse={false}
  aside={<div>侧边栏（不可折叠）</div>}
>
  <div>主内容</div>
</Layout>
```

**实际效果：**

<div style={{ border: '1px solid #e5e5e5', borderRadius: '4px', overflow: 'hidden', margin: '16px 0', height: '300px' }}>
  <Layout 
    mode="AB"
    isCanCollapse={false}
    aside={
      <div style={{ padding: '16px' }}>
        <h4 style={{ marginTop: 0 }}>侧边栏（不可折叠）</h4>

      </div>

    <div style={{ padding: '16px' }}>

</div>

## 注意事项

1. 布局模式通过 `mode` 属性控制，不同的模式会显示不同的区域
2. 侧边栏折叠功能默认开启，可以通过 `isCanCollapse={false}` 禁用
3. 默认使用全高度布局，适合单页应用场景
4. 侧边栏宽度支持数字（px）和字符串（如 '20%'）两种格式
